# 重排序与融合

## 概述

多路召回产生的检索结果需要经过融合与重排序，才能获得高质量的最终结果。本规范涵盖 RRF（Reciprocal Rank Fusion）融合策略、Cross-encoder 重排序、分数归一化方法及其 Python 实现。

## 核心规则

### 🔴 必须遵守 (MUST)

1. **多路召回结果必须融合排序**
   - 不同类型的检索器返回的结果无法直接比较分数
   - 必须使用 RRF 或类似的跨系统排序方法

2. **最终结果必须经过重排序**
   - 使用 Cross-encoder 或 LLM 对候选结果重新打分
   - 重排序阶段的候选数应为最终结果数的 2-4 倍

### 🟡 强烈建议 (SHOULD)

1. **RRF 作为默认融合策略**
   - RRF 不依赖分数绝对值，对分数分布差异鲁棒
   - 参数 k 默认值 60，可根据数据量调整

2. **分数归一化后再融合**
   - 向量检索分数范围 [0, 1]，BM25 分数范围 [0, 10+]
   - 归一化到 [0, 1] 后再融合

3. **Cross-encoder 重排序**
   - 使用 Cohere Rerank, BGE-Reranker 等专用重排序模型
   - 重排序 batch size 控制，避免 OOM

### 🟢 可选建议 (MAY)

1. **LLM 重排序**
   - 让 LLM 对候选结果进行相关性评分
   - 成本较高，适合高精度要求的场景

2. **自适应融合权重**
   - 根据历史反馈动态调整各路由的权重
   - 不同查询类型使用不同融合权重

## 正确示例

```python
import numpy as np
from typing import List
from dataclasses import dataclass


@dataclass
class RankedItem:
    content: str
    metadata: dict
    original_score: float
    source: str          # 来源检索器
    original_rank: int   # 在原检索器中的排名
    
    reranked_score: float = 0.0


class ScoreNormalizer:
    """分数归一化"""
    
    @staticmethod
    def min_max_normalize(scores: List[float]) -> List[float]:
        """Min-Max 归一化到 [0, 1]"""
        scores_arr = np.array(scores)
        min_val = scores_arr.min()
        max_val = scores_arr.max()
        if max_val == min_val:
            return [0.5] * len(scores)
        return ((scores_arr - min_val) / (max_val - min_val)).tolist()
    
    @staticmethod
    def z_score_normalize(scores: List[float]) -> List[float]:
        """Z-Score 归一化"""
        scores_arr = np.array(scores)
        mean = scores_arr.mean()
        std = scores_arr.std()
        if std == 0:
            return [0.0] * len(scores)
        return ((scores_arr - mean) / std).tolist()
    
    @staticmethod
    def rank_based_normalize(ranks: List[int], total: int) -> List[float]:
        """基于排名的归一化分数 = 1 - rank / total"""
        return [1 - r / total for r in ranks]


class RRFFuser:
    """RRF (Reciprocal Rank Fusion) 融合器"""
    
    def __init__(self, k: int = 60):
        """
        Args:
            k: RRF 常数参数，默认 60
               较小的 k 对排名靠前的结果给予更高权重
               较大的 k 使权重分布更平滑
        """
        self.k = k
    
    def fuse(self, items_by_source: dict[str, List[RankedItem]]) -> List[RankedItem]:
        """
        融合多路召回结果
        Args:
            items_by_source: {"vector": [...], "bm25": [...]}
        Returns:
            融合排序后的结果列表
        """
        # 按内容去重
        seen_contents: set[str] = set()
        all_items: List[RankedItem] = []
        
        for source, items in items_by_source.items():
            for item in items:
                content_key = item.content[:200]  # 用前缀做近似去重
                if content_key not in seen_contents:
                    seen_contents.add(content_key)
                    all_items.append(item)
        
        # 为每个来源维护排名
        source_ranks: dict[str, dict[str, int]] = {}
        for source, items in items_by_source.items():
            source_ranks[source] = {}
            for i, item in enumerate(items):
                content_key = item.content[:200]
                source_ranks[source][content_key] = i + 1  # 1-indexed
        
        # 计算 RRF 分数
        for item in all_items:
            content_key = item.content[:200]
            rrf_score = 0.0
            for source in items_by_source.keys():
                rank = source_ranks[source].get(content_key)
                if rank is not None:
                    rrf_score += 1 / (self.k + rank)
            item.reranked_score = rrf_score
        
        # 按 RRF 分数降序排列
        all_items.sort(key=lambda x: x.reranked_score, reverse=True)
        return all_items


class CrossEncoderReranker:
    """Cross-encoder 重排序器"""
    
    def __init__(self, model_name: str = "BAAI/bge-reranker-v2-m3"):
        self.model_name = model_name
        self.model = None  # 延迟加载
    
    def _load_model(self):
        """延迟加载模型"""
        if self.model is None:
            from transformers import AutoModelForSequenceClassification, AutoTokenizer
            self.tokenizer = AutoTokenizer.from_pretrained(self.model_name)
            self.model = AutoModelForSequenceClassification.from_pretrained(self.model_name)
    
    def rerank(self, query: str, candidates: List[RankedItem], 
               top_k: int = 5) -> List[RankedItem]:
        """
        使用 Cross-encoder 重排序
        Args:
            query: 原始查询
            candidates: 候选结果
            top_k: 返回数量
        Returns:
            重排序后的结果
        """
        self._load_model()
        
        # 准备 query-document pairs
        pairs = [(query, item.content) for item in candidates]
        
        # Batch 推理
        batch_size = 16
        all_scores = []
        
        for i in range(0, len(pairs), batch_size):
            batch_pairs = pairs[i:i + batch_size]
            inputs = self.tokenizer(
                batch_pairs,
                padding=True,
                truncation=True,
                max_length=512,
                return_tensors="pt",
            )
            outputs = self.model(**inputs)
            scores = outputs.logits.squeeze(-1).detach().numpy()
            all_scores.extend(scores.tolist())
        
        # 更新分数并排序
        for item, score in zip(candidates, all_scores):
            item.reranked_score = score
        
        candidates.sort(key=lambda x: x.reranked_score, reverse=True)
        return candidates[:top_k]


class FusionPipeline:
    """完整融合管道：多路召回 → RRF 融合 → Cross-encoder 重排序"""
    
    def __init__(self, fuser: RRFFuser, reranker: CrossEncoderReranker | None = None):
        self.fuser = fuser
        self.reranker = reranker
    
    async def process(self, query: str, 
                      vector_results: List[RankedItem],
                      bm25_results: List[RankedItem]) -> List[RankedItem]:
        """
        处理检索结果
        Step 1: RRF 融合
        Step 2: Cross-encoder 重排序（可选）
        """
        # Step 1: RRF 融合
        items_by_source = {
            "vector": vector_results,
            "bm25": bm25_results,
        }
        fused = self.fuser.fuse(items_by_source)
        
        print(f"[Fusion] RRF 融合完成: {len(vector_results)} 向量 + "
              f"{len(bm25_results)} BM25 → {len(fused)} 去重结果")
        
        # Step 2: Cross-encoder 重排序
        if self.reranker and len(fused) > 5:
            reranked = self.reranker.rerank(query, fused, top_k=5)
            print(f"[Fusion] Cross-encoder 重排序完成: {len(fused)} → {len(reranked)}")
            return reranked
        
        return fused[:5]
```

## 错误示例

```python
# 错误 1：直接比较不同检索器的原始分数
def bad_direct_score_comparison():
    """向量余弦相似度 0.85 和 BM25 分数 12.3 无法直接比较"""
    vector_score = 0.85  # 范围 [0, 1]
    bm25_score = 12.3    # 范围 [0, ~20+]
    # 如果直接排序，BM25 结果始终排前面
    # 但实际上 0.85 的余弦相似度可能比 12.3 的 BM25 更相关


# 错误 2：不使用 RRF，只用加权平均
def bad_weighted_average():
    """加权平均需要分数分布一致，否则结果不可靠"""
    vector_results = [0.9, 0.8, 0.7]  # 分布集中
    bm25_results = [15.0, 3.0, 1.0]   # 分布分散
    # 加权平均后 BM25 的异常高分主导排序
    # RRF 对分布不敏感，更适合这类场景


# 错误 3：不重排序直接拼接
def bad_concatenate():
    """多路召回结果直接拼接"""
    results = []
    results.extend(vector_results)  # 前 10 个
    results.extend(bm25_results)    # 后 10 个
    # vector_results 的最后几名质量可能低于 bm25_results 的前几名
    # 需要融合重排


# 错误 4：重排序候选集太多
def bad_too_many_candidates():
    """Cross-encoder 处理 1000 个候选 → 延迟过高"""
    candidates = get_all_candidates(top_k=1000)
```

## 工具链配置

```python
# BGE-Reranker 使用示例
def create_reranker_pipeline() -> FusionPipeline:
    fuser = RRFFuser(k=60)
    
    # 如果 GPU 可用，使用 Cross-encoder
    import torch
    if torch.cuda.is_available():
        reranker = CrossEncoderReranker("BAAI/bge-reranker-v2-m3")
    else:
        reranker = None  # 无 GPU 时跳过 Cross-encoder
    
    return FusionPipeline(fuser=fuser, reranker=reranker)
```
