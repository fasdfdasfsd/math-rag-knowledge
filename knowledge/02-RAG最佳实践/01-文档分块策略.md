# 文档分块策略

## 概述

文档分块（Chunking）是 RAG 系统的关键预处理步骤。合理的分块策略直接影响检索精度和生成质量。本规范对比常见分块方法，提供默认推荐策略，并针对特殊文档类型给出最佳实践。

## 核心规则

### 🔴 必须遵守 (MUST)

1. **默认使用递归字符分块**
   - `chunk_size=512` tokens，`chunk_overlap=50`
   - 分隔符优先级：`\n\n` > `\n` > 句号 > 逗号 > 空格

2. **每个 chunk 必须保留完整元数据**
   - 必须包含：`source`（来源文件）、`page`（页码）、`section`（章节标题）、`title`（文档标题）
   - 可选包含：`author`、`date`、`tags`、`summary`

3. **特殊文档类型使用专用分块器**
   - 代码文件：使用 AST（抽象语法树）分块，保持函数/类完整性
   - 表格数据：保持表格完整，不跨行拆分

### 🟡 强烈建议 (SHOULD)

1. **根据文档类型调整 chunk_size**
   - 技术文档：384-512 tokens
   - 代码文件：256-384 tokens
   - 对话记录：128-256 tokens（保持单轮对话完整）
   - 书籍/长文：768-1024 tokens

2. **语义分块作为备选**
   - 对主题跳跃明显的文档，考虑使用语义分块
   - 嵌入了 embedding 相似度分割

3. **分块质量验证**
   - 检查：每个 chunk 在脱离上下文后是否仍然有意义
   - 检查：关键信息没有被跨 chunk 拆分
   - 检查：chunk 边界没有截断重要术语或代码

### 🟢 可选建议 (MAY)

1. **文档结构分块**
   - 对 Markdown/HTML 文档，按标题层级分块
   - 保留文档大纲结构

2. **自适应分块**
   - 根据文档长度自动调整 chunk_size
   - 短文档使用更小的 chunk_size

## 正确示例

```python
from typing import List, Optional
from dataclasses import dataclass, field
import re


@dataclass
class DocumentChunk:
    text: str
    metadata: dict
    chunk_id: str
    
    @classmethod
    def create(cls, text: str, source: str, page: int = 0, 
               section: str = "", title: str = "", **extra) -> "DocumentChunk":
        return cls(
            text=text,
            metadata={
                "source": source,
                "page": page,
                "section": section,
                "title": title,
                **extra
            },
            chunk_id=f"{source}::p{page}::s{section}::{hash(text) % 10000}"
        )


class RecursiveCharacterChunker:
    """递归字符分块器 — 默认推荐实现"""
    
    SEPARATORS = ["\n\n", "\n", "。", "，", " ", ""]
    
    def __init__(self, chunk_size: int = 512, chunk_overlap: int = 50):
        self.chunk_size = chunk_size
        self.chunk_overlap = chunk_overlap
    
    def split_text(self, text: str) -> List[str]:
        """递归分割文本"""
        return self._recursive_split(text, self.SEPARATORS)
    
    def _recursive_split(self, text: str, separators: List[str]) -> List[str]:
        """递归分割实现"""
        final_chunks: List[str] = []
        separator = separators[0]
        new_separators = separators[1:]
        
        if separator:
            splits = text.split(separator)
        else:
            splits = list(text)
        
        good_splits: List[str] = []
        for s in splits:
            if len(s) < self.chunk_size:
                good_splits.append(s)
            else:
                if good_splits:
                    merged = self._merge_splits(good_splits)
                    final_chunks.extend(merged)
                    good_splits = []
                if new_separators:
                    final_chunks.extend(self._recursive_split(s, new_separators))
                else:
                    final_chunks.append(s)
        
        if good_splits:
            merged = self._merge_splits(good_splits)
            final_chunks.extend(merged)
        
        return final_chunks
    
    def _merge_splits(self, splits: List[str]) -> List[str]:
        """将小段合并为 chunk"""
        chunks: List[str] = []
        current: List[str] = []
        current_size = 0
        
        for s in splits:
            s_size = len(s)
            if current_size + s_size > self.chunk_size:
                if current:
                    chunks.append("".join(current))
                    overlap_text = self._get_overlap(current)
                    current = [overlap_text] if overlap_text else []
                    current_size = len(overlap_text) if overlap_text else 0
            current.append(s)
            current_size += s_size
        
        if current:
            chunks.append("".join(current))
        
        return chunks
    
    def _get_overlap(self, current_chunk: List[str]) -> str:
        """从当前 chunk 末尾提取 overlap 文本"""
        overlap_text = ""
        overlap_chars = 0
        for s in reversed(current_chunk):
            if overlap_chars + len(s) > self.chunk_overlap:
                remaining = self.chunk_overlap - overlap_chars
                if remaining > 0:
                    overlap_text = s[-remaining:] + overlap_text
                break
            overlap_text = s + overlap_text
            overlap_chars += len(s)
        return overlap_text


class ASTCodeChunker:
    """基于 AST 的代码分块"""
    
    def __init__(self, language: str = "python", chunk_size: int = 256):
        self.language = language
        self.chunk_size = chunk_size
    
    def chunk_code(self, code: str, source: str) -> List[DocumentChunk]:
        """按 AST 节点分块代码文件"""
        import ast
        
        chunks: List[DocumentChunk] = []
        
        try:
            tree = ast.parse(code)
        except SyntaxError:
            # 解析失败时回退到行级分块
            return self._fallback_line_chunk(code, source)
        
        for node in ast.walk(tree):
            if isinstance(node, (ast.FunctionDef, ast.AsyncFunctionDef, ast.ClassDef)):
                start_line = node.lineno
                end_line = node.end_lineno
                node_text = self._extract_lines(code, start_line, end_line)
                
                chunk_text = (
                    f"```{self.language}\n"
                    f"{node_text}\n"
                    f"```\n"
                    f"Source: {source} (line {start_line}-{end_line})"
                )
                
                chunks.append(DocumentChunk.create(
                    text=chunk_text,
                    source=source,
                    section=f"{node.name}",
                    title=f"{type(node).__name__}: {node.name}",
                    start_line=start_line,
                    end_line=end_line,
                ))
        
        return chunks
    
    @staticmethod
    def _extract_lines(code: str, start: int, end: int) -> str:
        lines = code.split("\n")
        return "\n".join(lines[start-1:end])
    
    @staticmethod
    def _fallback_line_chunk(code: str, source: str) -> List[DocumentChunk]:
        """语法解析失败时的行级回退"""
        lines = code.split("\n")
        chunks = []
        for i in range(0, len(lines), 20):
            chunk_text = "\n".join(lines[i:i+20])
            chunks.append(DocumentChunk.create(
                text=chunk_text,
                source=source,
                section=f"line {i+1}",
                title=f"Code snippet from {source}",
            ))
        return chunks
```

## 错误示例

```python
# 错误 1：chunk_size 过大
def bad_large_chunk():
    """单 chunk 5000 tokens → 检索精度严重下降"""
    chunker = RecursiveCharacterChunker(chunk_size=5000, chunk_overlap=200)
    # 一个 chunk 包含多章内容
    # 检索时无法精确定位到具体段落
    # LLM 生成时关注力分散在大量不相关内容上


# 错误 2：chunk_size 过小
def bad_small_chunk():
    """单 chunk 32 tokens → 上下文完全丢失"""
    chunker = RecursiveCharacterChunker(chunk_size=32, chunk_overlap=4)
    text = "FastAPI 是一个现代、快速（高性能）的 Web 框架"
    chunks = chunker.split_text(text)
    # 结果：["FastAPI 是一个现代、快速", "（高性能）的 Web 框架"]
    # "FastAPI" 和 "Web 框架" 不在同一个 chunk 中
    # 检索 "Web 框架" 时无法关联到 "FastAPI"


# 错误 3：不保留元数据
def bad_no_metadata():
    """检索后无法溯源"""
    chunks = [
        {"text": "FastAPI 是一个 Web 框架"}
        # 没有 source/page/section 信息
        # 用户问 "这个结论出自哪里？" → 无法回答
    ]


# 错误 4：代码文件按行分块
def bad_line_chunk_code():
    """代码按行分块 → 函数被切断"""
    code = """
def calculate_total(items):
    total = 0
    for item in items:
        total += item.price
    return total

def apply_discount(total, rate):
    return total * (1 - rate)
"""
    chunks = code.split("\n")  # 按行分割
    # chunk 3: "    total += item.price"
    # chunk 4: "    return total"
    # 一个函数的代码分散在多个 chunk
    # 检索语义完全丢失


# 错误 5：无 overlap
def bad_no_overlap():
    """chunk_overlap=0 → 边界信息丢失"""
    chunker = RecursiveCharacterChunker(chunk_size=100, chunk_overlap=0)
    # "激活函数如 ReLU 和 Sigmoid，其中 ReLU 在深层网络表现更好"
    # chunk 1 结尾: "...激活函数如 ReLU"
    # chunk 2 开头: "在深层网络表现更好"
    # "和 Sigmoid" 可能丢失，"ReLU" 的完整描述被切断
```
