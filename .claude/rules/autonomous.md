# 无人值守执行规则

## 启动即执行

本文件存在 = 用户要求无人值守模式。不要问"是否开始"、"是否继续"。

## 执行节奏

1. **输入预处理** — 每轮收到用户输入 → 理解意图 → 分析归因 → 管家调度 → 输出反馈（详见 CLAUDE.md §输入预处理协议）
2. 读取 activeContext.md → 确定当前阶段和待办
3. 按 11 阶段流程推进（不跳过、不逆向）
4. 每阶段派遣 ≥3 个智能体 → 共识 → 汇报
5. 需要决策 → 选择题（选项+推荐+理由）
6. 上下文 75% → compact
7. 阶段性成果 → 更新 MEMORY.md + activeContext.md
8. **退出前 → 检查并更新所有文档和知识库（primer.md / 避坑汇总 / session-log / 规范文件）**

## 禁止词汇

以下词汇在无人值守模式下禁止出现：
- "需要我继续吗？"
- "你同意吗？"
- "要不要...？"
- "我应该...吗？"
- "你想让我...吗？"
- "接下来做什么？"
- "请确认..."
- "是否...？"

## 替代表达

| 禁止 | 替代 |
|------|------|
| "需要我继续吗？" | "继续执行：{具体动作}" |
| "你同意吗？" | "以上方案（推荐 A），理由：...。如无异议我将执行。" |
| "接下来做什么？" | "下一步：{具体步骤}（预估 X min）" |
