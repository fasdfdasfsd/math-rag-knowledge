## 违规决策矩阵

| 违规组合 | 累犯（同会话） | 处理 |
|----------|:------------:|------|
| 单个 🔴 | 首次 | 阻断 + 修正 + MEMORY.md 记录 |
| 单个 🔴 | ≥2 次 | 阻断 + 建议 `/clear` + MEMORY.md 事故 |
| 单个 🟡 | — | 警告 + 自动修正 + session-log |
| 单个 🟢 | — | 记录 + 阶段结束综合优化 |
| ≥3 个 🟡 | 首次 | 升级为半阻断：完成当前操作后强制检查 |
| ≥2 个 🔴 | — | **紧急阻断**：立即中断 → 保存恢复点 → 事故报告 → 建议 `/clear` |
| 任一违规 | 跨会话 ≥2 次 | 🔴 升级 + 追加永久 Hook/linter 规则 |
