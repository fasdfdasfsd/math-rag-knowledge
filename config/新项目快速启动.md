# 新项目快速启动

> 从零创建一个继承 v4.2 完整工程能力的新项目
> 版本：v4.2 | 日期：2026-05-31

---

## 一、快速启动（5 分钟）

```bash
# 1. 创建新项目
mkdir my-project && cd my-project && git init

# 2. 复制通用资源
cp -r /path/to/python-rag/knowledge ./knowledge
cp -r /path/to/python-rag/config ./config
cp -r /path/to/python-rag/deploy ./deploy
cp /path/to/python-rag/.claudeignore ./.claudeignore
cp /path/to/python-rag/.gitignore ./.gitignore

# 3. 一键部署
chmod +x deploy/install.sh && ./deploy/install.sh
```

## 二、部署后必做（4 项修改）

### 2.1 编辑 CLAUDE.md

```diff
- **项目名称**：python-rag（检索增强生成系统）
- **技术栈**：Python 3.12+ / FastAPI / PostgreSQL + pgvector / Redis
+ **项目名称**：my-project（<项目描述>）
+ **技术栈**：<你的技术栈>

- | 简单任务 | deepseek-v4-flash | Agent (默认) |
- | 复杂推理 | deepseek-v4-pro | Agent (opus) |
+ | <你的模型分级> |
```

### 2.2 重置动态记忆

```bash
# primer.md — 清空旧 ADR，保留模板结构
echo "# primer.md — 项目架构约定

> 最后更新: $(date +%Y-%m-%d)
> 性质: 动态记忆层，随架构变更更新

## 项目身份

- **项目名称**：my-project
- **技术栈**：<填写>
- **包管理器**：<uv/npm/pnpm>
- **代码检查**：<Ruff/ESLint>
- **类型检查**：<mypy/TypeScript>
- **测试框架**：<pytest/jest>

## 架构决策记录

### ADR-001: <第一个架构决策>

## 关键约定

（从 CLAUDE.md 提取）" > .claude/primer.md

# activeContext.md — 重置为空白模板
cp config/L2-项目共享/activeContext.md ./.claude/activeContext.md

# MEMORY.md — 重置为空白模板
cp config/L2-项目共享/MEMORY.md ./.claude/MEMORY.md
```

### 2.3 规则文件按需选择

```bash
mkdir -p .claude/rules

# 以下规则通用 — 直接复制
cp /path/to/python-rag/.claude/rules/security.md ./.claude/rules/

# 以下规则按技术栈选择
# Python 项目:  cp python.md + api.md + database.md + testing.md
# JS/TS 项目:  创建 .claude/rules/typescript.md (参考 python.md 结构)
# 数据库:      用了就复制 database.md
```

### 2.4 重新安装 MCP

MCP 配置在 `~/.claude.json`，按项目路径关联，**不能随文件夹复制**。

```bash
# 按 v4.2 策略选择安装（参考避坑 #13~#19）
# P0 常驻 (3 件)
claude mcp add git -- npx -y git-mcp-server
claude mcp add github -- npx -y @modelcontextprotocol/server-github
claude mcp add filesystem -- npx -y @modelcontextprotocol/server-filesystem .

# P0 触发 (5 件，按需)
claude mcp add search -- npx -y one-search-mcp
claude mcp add playwright -- npx @playwright/mcp@latest
# ... 其余参见 deploy/v4.2-MCP部署手册.md
```

## 四、MCP 安装命令速查

```bash
# 语法: claude mcp add <name> [选项] -- npx -y <package> [参数]

# 安全
# (EdgeBox 或 Docker 直接沙箱 — 见部署手册)

# 版本控制
claude mcp add git -- npx -y git-mcp-server
claude mcp add github -- npx -y @modelcontextprotocol/server-github

# 知识获取
claude mcp add search -- npx -y one-search-mcp                 # 多引擎搜索
claude mcp add playwright -- npx @playwright/mcp@latest        # 浏览器
npx playwright install chromium

# 数据操作
claude mcp add postgres -- npx -y @modelcontextprotocol/server-postgres \
  "postgresql://readonly_user:pass@localhost:5432/dbname"
claude mcp add redis -- npx -y @gongrzhe/server-redis-mcp redis://localhost:6379

# 效率增强
claude mcp add filesystem -- npx -y @modelcontextprotocol/server-filesystem .
claude mcp add docker -- npx -y mcp-server-docker

# Token 成本优化配置
export CLAUDE_AUTOCOMPACT_PCT_OVERRIDE=80
export CLAUDE_CODE_MAX_CONTEXT_TOKENS=100000
export CLAUDE_CODE_DISABLE_FAST_MODE=1
```
