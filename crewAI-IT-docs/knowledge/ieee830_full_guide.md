# 标题：百度amis低代码需求设计和功能描述文档

## 一：需求设计



# Amis 低代码框架全面需求说明书

依据官方能力体系，对应拆解**业务需求、功能需求、技术需求、体验需求、部署集成需求、运维生态需求**全维度明细，粒度匹配前文超细功能规格

## 一、总体业务需求

1. 快速搭建企业中后台管理系统，覆盖权限、数据管理、流程审批、数据看板、日志监控通用业务场景
2. 降低前后端开发成本，减少 HTML/CSS/JS 手写编码，缩短页面交付周期
3. 适配多岗位使用，支持开发人员配置编写、产品 / 运维可视化拖拽搭建页面
4. 满足业务频繁迭代变更，配置修改即可更新页面，无需复杂编译发布
5. 统一全站 UI 视觉风格，规避页面样式杂乱问题
6. 实现业务数据录入、查询、编辑、删除、导出、统计可视化全链路管理

## 二、核心引擎能力需求

### 2.1 配置渲染需求

1. 支持纯 JSON Schema 描述页面，自动解析生成完整可交互页面
2. 无前端框架强绑定，可脱离 React/Vue 独立运行，兼容多技术宿主环境
3. 配置版本向下兼容，历史旧配置可在新版本正常渲染运行
4. 渲染链路稳定，JSON 转虚拟 DOM 再映射真实 DOM，自动绑定样式、事件、数据

### 2.2 表达式动态逻辑需求

1. 支持页面内全局、局部、嵌套字段变量读取引用
2. 具备算术、比较、逻辑、三元运算能力，支撑数值计算与条件判断
3. 内置文本、数值、日期、数组、逻辑类通用工具函数
4. 可通过表达式动态控制文本内容、样式、组件显隐、禁用、只读状态

### 2.3 可视化编辑需求

1. 提供拖拽式画布，支持组件拖拽添加、位置调整、层级排布、对齐排版
2. 拖拽操作与 JSON 配置双向实时同步，修改视图同步更新源码
3. 配套可视化属性面板，可视化修改组件基础属性、外观、数据、事件
4. 内置多类业务页面模板，支持模板复用、页面复制、JSON 导入导出

### 2.4 混合扩展开发需求

1. 内置足量通用组件，覆盖绝大多数常规业务页面搭建
2. 支持基于 React 开发自定义业务组件，无缝嵌入框架体系
3. 自定义组件共用框架数据域、事件、权限、表达式能力，支持双向数据交互
4. 组件支持无限层级嵌套组合，可搭建复杂多层级页面结构

## 三、页面布局类功能需求

1. 具备页面根容器能力，可统一管控页面标题、边距、背景、全屏、头尾固定样式

## 二：功能清单：

### 百度 Amis 官方功能全集（超细粒度・完全版）

### 一、核心架构与引擎能力（底层机制）

### 1. JSON Schema 驱动渲染（核心）

* **纯配置驱动**：页面结构、组件树、属性、事件、数据逻辑全部由 JSON 描述，**零手写 HTML/CSS/JS**。
* **无框架依赖**：不强制依赖 React/Vue/Angular，可直接嵌入任意网页（JSP/PHP/ 静态 HTML）。
* **配置即代码**：
  * 一个标准 CRUD 页面约 **150–200 行 JSON**
  * 开发周期由 3–5 天 → **0.5–1 天**
  * 代码量减少 **80%+**
* **版本强兼容**：**6 年前的配置当前版本可直接运行**，无需改造。
* **渲染流程**：JSON → 虚拟 DOM → 真实 DOM，自动处理绑定、事件、样式。

### 2. 表达式引擎（amis-formula）—— 动态能力核心

**支持在所有组件属性中使用 `${...}` 表达式**

* **变量取值**：
  * `${name}`：取当前数据域字段
  * `${user.name}`：嵌套对象
  * `${item}`：循环内当前项
* **运算能力**：
  * 算术：`+ - * / %`
  * 比较：`> < >= <= == !=`
  * 逻辑：`&& || !`
  * 三元：`${age>18 ? '成年' : '未成年'}`
* **内置函数（官方完整）**
  * 文本：`CONCAT、SUBSTRING、LENGTH、UPPERCASE、LOWERCASE、TRIM`
  * 数值：`SUM、AVG、MAX、MIN、ROUND、CEIL、FLOOR、ABS`
  * 日期：`NOW、DATEFORMAT、DATEDIFF、TIMESTAMP、YEAR、MONTH、DAY`
  * 数据：`GET、SET、PICK、OMIT、ARRAY_MAP、ARRAY_FILTER`
  * 逻辑：`IF、AND、OR、NOT、XOR、IS_EMPTY、IS_NOT_EMPTY`
* **动态控制**：
  * 动态文本、动态 class、动态 style
  * 动态显隐（`visibleOn`）、动态禁用（`disabledOn`）
  * 动态只读、动态占位符、动态选项

### 3. 可视化编辑器（amis-editor）—— 零代码搭建

* **拖拽布局**：左侧组件树拖拽到画布，自由摆放、对齐、层级调整。
* **实时预览**：所见即所得，**拖拽即渲染**，无需刷新。
* **双向同步**：拖拽操作 ↔ JSON 配置实时同步，**改配置即改界面**。
* **属性面板（超细）**：
  * 基础：类型、ID、名称、标签、描述、提示
  * 外观：尺寸、颜色、圆角、边距、背景、边框、阴影
  * 状态：显隐、禁用、只读、加载
  * 数据：绑定字段、默认值、数据源、接口
  * 事件：点击、变更、提交、加载完成等动作配置
* **模板系统**：
  * 内置 50+ 常用页面模板（登录页、CRUD、表单页、仪表盘）
  * 支持**自定义模板保存 / 复用 / 分享**
* **页面管理**：新建、复制、删除、重命名、导出 JSON、导入 JSON

### 4. 混合开发模式（90% 低代码 + 10% 代码扩展）

* **原生组件全覆盖**：内置 **120+ 官方组件**，满足 90% 中后台场景。
* **自定义组件（React）**：
  * 基于 React 开发，**无缝嵌入 amis 配置**
  * 共享 amis **数据域、事件系统、权限控制、表达式**
  * 支持双向绑定、事件触发、数据回传
* **无限级嵌套**：组件可**任意深度嵌套**（Page → Grid → Panel → Tabs → Form → Input）。
* **作为 UI 库使用**：可单独引用 amis 组件，在 React/Vue 项目中直接使用。

## 二、布局组件（页面结构・超细分类）

### 1. 页面容器类

#### Page（页面根容器）

* 配置项：标题、副标题、页面内边距、外边距、背景色、全屏模式、头部固定、底部固定
* 功能：统一页面样式、控制全局布局、支持路由绑定

#### Container（基础容器）

* 功能：包裹内容、控制边距、背景、边框、阴影、圆角
* 支持：`className`、`style`、`visibleOn`、`disabledOn`

### 2. 栅格布局类

#### Grid（栅格）

* 配置：行列数、 gutter（间距）、列宽比例（如 1:2:1）、响应式断点（手机 / 平板 / PC）
* 功能：自动均分或按比例分配宽度，**响应式自动换行**

#### HBox（水平排列）

* 功能：子组件水平排列，支持间距、对齐方式（左 / 中 / 右 / 两端）、自动换行

#### VBox（垂直排列）

* 功能：子组件垂直排列，支持间距、对齐方式（上 / 中 / 下）、高度自适应

### 3. 面板类

#### Panel（卡片面板）

* 配置：标题、副标题、图标、可折叠、默认展开、头部样式、内容样式、底部按钮
* 功能：分组展示内容、支持折叠 / 展开、自带标题栏

#### Collapse（折叠面板组）

* 功能：多个 Panel 组合，**手风琴模式（同时仅一个展开）** 或 独立模式
* 配置：默认展开项、折叠图标、展开图标

### 4. 标签页类

#### Tabs（选项卡）

* 配置：标签位置（上 / 下 / 左 / 右）、标签样式（默认 / 卡片 / 线条）、默认激活项、懒加载（仅激活时渲染）
* 功能：多内容切换、减少页面空间占用

#### Tab（单个标签页）

* 配置：标题、图标、禁用、隐藏、内容

### 5. 导航布局类

#### Sidebar（侧边栏）

* 配置：宽度、折叠宽度、可折叠、默认折叠、菜单数据源、图标、层级
* 功能：左侧 / 右侧导航菜单、支持路由跳转、折叠收缩

#### Header（顶部导航）

* 配置：高度、背景色、logo、标题、导航菜单、用户信息、退出按钮
* 功能：页面头部、全局导航入口

#### Footer（底部栏）

* 配置：高度、背景色、版权信息、链接
* 功能：页面底部、版权声明

#### Breadcrumb（面包屑）

* 配置：数据源、分隔符、路由跳转
* 功能：显示当前页面层级路径、快速返回上级

#### Steps（步骤条）

* 配置：步骤数、当前步骤、步骤状态（等待 / 进行中 / 完成 / 错误）、方向（水平 / 垂直）
* 功能：流程进度展示、步骤导航

### 6. 弹窗 / 浮层类

#### Modal（模态弹窗）

* 配置：标题、宽度、高度、全屏、可拖拽、可关闭、遮罩、关闭按钮、位置（居中 / 顶部 / 底部）
* 功能：弹出层展示内容、表单、详情、确认操作

#### Drawer（抽屉）

* 配置：方向（左 / 右 / 上 / 下）、宽度 / 高度、可关闭、遮罩、关闭按钮
* 功能：侧边滑出层、适合详情页、表单页

#### Affix（固钉）

* 配置：偏移量（top/bottom）、滚动容器
* 功能：滚动时固定在页面指定位置（如回到顶部按钮、操作栏）

### 7. 分割布局类

#### Split（分割面板）

* 配置：方向（水平 / 垂直）、默认比例、最小尺寸、可拖拽
* 功能：左右 / 上下分栏、可拖拽调整大小（如左侧列表 + 右侧详情）

## 三、表单组件（数据录入・120 + 项・超细）

### 1. 表单容器

#### Form（表单）

* 核心配置：
  * `mode`：normal（垂直）/horizontal（水平）/inline（行内）

* **基础验证**：`required`（必填）、`maxLength`、`minLength`、`max`、`min`
* **正则验证**：`pattern`（自定义正则）、`type`（email/phone/url/idcard）
* **异步验证**：`validateApi`（后台实时校验，如用户名是否存在）
* **联动验证**：根据其他字段值动态验证（如密码一致性）
* **错误提示**：自定义 `message`、全局错误展示位置、是否隐藏详情
* **验证时机**：`validateOnChange`（变更即验证）、`validateOnSubmit`（提交时验证）

### 6. 表单联动

* **显隐联动**：`visibleOn`（如选择 “其他” 才显示输入框）
* **禁用联动**：`disabledOn`（如未勾选协议则禁用提交）
* **赋值联动**：`valueOn`（如选择省份后自动赋值默认城市）
* **选项联动**：`source` 接口携带其他字段值（如省份变更后重新获取城市列表）

## 六、企业级能力（权限 / 国际化 / 主题 / 性能）

### 1. 权限控制（超细粒度）

* **页面级权限**：页面是否可访问
* **组件级权限**：按钮显隐、表单禁用、面板隐藏
* **列级权限**：表格字段不同角色可见 / 隐藏
* **数据级权限**：接口过滤数据（如只能看自己创建的数据）
* **权限集成**：支持 Token、Cookie、Header 权限校验、自定义权限函数

### 2. 国际化（i18n）

* **全局语言包**：内置中文、英文，支持扩展其他语言
* **配置方式**：使用 `${t('key')}` 关联语言包
* **运行时切换**：一键切换语种，**无需刷新页面**
* **覆盖范围**：按钮文案、表单提示、弹窗文案、错误信息、组件默认文案

### 3. 主题定制

* **内置主题**：
  * `cxd`（云舍）：默认主题，简约现代
  * `antd`：仿 Ant Design 风格
* **自定义主题**：
  * 主色调、辅助色、文字色、背景色
  * 字体、字号、行高
  * 圆角、间距、边框、阴影
* **全局生效**：一次配置，所有组件自动应用

### 4. 性能优化（官方内置）

* **数据缓存**：接口请求结果缓存，**避免重复请求**，可配置缓存时长
* **虚拟滚动**：大数据表格 / 列表（1000 + 行）仅渲染可视区域，**解决卡顿**
* **组件懒加载**：Tab、Modal、Drawer 内组件**仅激活时渲染**，减少首屏加载时间
* **资源按需加载**：JS/CSS 分片加载，首屏体积最小化
* **防抖节流**：高频操作（如搜索、滚动）自动防抖，**防止频繁请求**
* **数据懒渲染**：非可视区域内容延迟渲染，提升首屏速度
