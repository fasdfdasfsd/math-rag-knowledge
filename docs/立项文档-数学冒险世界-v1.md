## 1. 项目基本信息

- **项目名称**：数学冒险世界（Math Adventure World）
- **项目编号**：MAW-2026-001
- **产品定位**：面向 6-14 岁儿童的 AI 驱动游戏化数学学习平台
- **产品负责人**：用户（PO）
- **技术负责人**：用户 + Claude Code AI（TL）
- **拟稿日期**：2026-06-01

---

## 2. 项目背景与目标

### 业务背景
- 全球游戏化数学 App 市场 2024 年约 28 亿美元，预计 2033 年达 111 亿美元（CAGR 21.4%）
- 中国 K-12 在线教育 2025 年市场规模约 4200 亿人民币
- 家长付费意愿明确：70%+ 愿意为优质教育科技产品付费
- 现有 23 款竞品中，**没有任何产品**实现"数学概念拟人化 + AI 冒险叙事 + 自适应难度"三位一体

### 解决的问题
- 儿童学数学"做题感"强，缺乏内在动机
- 现有产品要么有游戏无叙事（Prodigy），要么有角色无自适应（Numberblocks）
- AI 讲题已成红海（好未来/猿辅导/作业帮），但 AI 驱动的数学叙事冒险是蓝海

### 预期目标
- **MVP（2.5-3 个月）**：13 项 Must-have 功能，支持学生完成完整六段式冒险关卡
- **运营指标**：D1 留存 > 70%，D7 > 40%，D30 > 20%
- **盈亏平衡**：12 个付费用户 @ ¥30/月

---

## 3. 范围

### In Scope
- 三层解耦架构（决策引擎 + RAG 约束层 + LLM 生成层）
- 六段式关卡叙事模板（入口→到达→冲突→解题→胜利→钩子）
- 数学概念拟人化 NPC 系统（≥ 30 个角色）
- 自适应难度引擎（滑动窗口 v2.0 → IRT/BKT v2.1）
- 知识图谱增强 RAG 检索
- 三层记忆架构（对话/经验/世界）
- Web 前端（React + TypeScript + PWA）

### Out of Scope（v2.0）
- 多人在线协作/竞争
- 自建语音识别（ASR）
- 移动原生 App（iOS/Android）
- LMS 深度集成

### 约束条件
- 时间：MVP 50-55 日历天
- 预算：首期 ¥5K（LLM API + 云服务器 + 顾问）
- 团队：1 人 + Claude Code AI 多智能体协作
- 合规：PIPL（中国未成年人数据保护）

## 5. 技术选型方案

| 组件 | 选型 | 决策依据 |
|------|------|----------|
| 后端框架 | Python 3.13 + FastAPI | 轻量异步，已有骨架 |
| 向量数据库 | Milvus（pymilvus） | 百万级向量，原生 HNSW |
| 关系数据库 | PostgreSQL 16 | 知识图谱 + 用户数据持久化 |
| 缓存 | Redis 7 | 热缓存 + 三层记忆热层 |
| LLM | DeepSeek Chat API（主），通义千问（备） | 兼容 OpenAI SDK，性价比高 |
| 前端 | React + TypeScript + Framer Motion | 游戏化动画生态最优 |
| 包管理 | uv（禁止 pip） | 项目规范 |
| 代码检查 | Ruff + mypy strict | 项目规范 |
| 测试 | pytest + pytest-asyncio | 项目规范 |

> 决策 ADR 详见 `.claude/primer.md`（ADR-001~009）
