## L2 Hooks × 9（11 条目）

| # | Hook 类型 | matcher | 行为 | 设计意图 |
|---|------|------|------|------|
| H-1 | PreToolUse | `rm -rf` | **阻止** (exit 2) | 防止不可逆递归删除 |
| H-2 | PreToolUse | `git push.*main` | **阻止** | 主分支必须通过 PR |
| H-3 | PreToolUse | `git push.*master` | **阻止** | 同上（master 分支） |
| H-4 | PreToolUse | `git commit` | **提醒** (exit 0) | 提交前执行质量门禁 |
| H-5 | PostToolUse | `Edit` | 提醒（no-op） | ruff format（待启用） |
| H-6 | PostToolUse | `Write` | 提醒（no-op） | 同上 |
| H-7 | Notification | `permission_prompt` | 桌面通知 | 提醒用户确认许可 |
| H-8 | Notification | `idle_prompt` | 桌面通知 | 提醒用户 Claude 等待 |
| H-9 | PreCompact | — | 提醒 | 压缩前保存记忆 |
| H-10 | Stop | — | 记录时间戳 + git diff + 提醒 | 检测异常退出 |
| H-11 | SessionStart | — | 记忆健康 + 项目健康摘要 | 启动即知状态 |
